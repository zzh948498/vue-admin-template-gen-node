## Description

[Zerone](https://github.com/zeronejs/zerone)

## Installation

```bash
$ pnpm i 
```

## Running the app

```bash
# development
$ pnpm dev

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
